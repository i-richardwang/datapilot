# v0.8.7 — Granular Feature Flags & Localization

## Features
- **Granular feature flags** — Replaced the monolithic `LITE_VERSION` switch with 5 independent build-time flags: `DATAPILOT_DISABLE_OAUTH`, `DATAPILOT_DISABLE_BROWSER`, `DATAPILOT_DISABLE_VALIDATION`, `DATAPILOT_DISABLE_TEMPLATES`, and `DATAPILOT_LITE_UI`. Each flag independently controls its tool set, system prompt sections, and UI elements.
- **OAuth disabled by default** — All builds (Electron, Docker, Zeabur) now ship with OAuth disabled and streamlined Lite UI enabled. Set `DATAPILOT_DISABLE_OAUTH=0` or `DATAPILOT_LITE_UI=0` to re-enable.
- **Hungarian, German, and Polish translations** — Three new languages added with full coverage of UI strings (synced from upstream). Polish includes proper plural form support.
- **API token refresh endpoint** — API-type sources can now specify an optional `renewEndpoint` for automatic token refresh without requiring full OAuth.

## Improvements
- **Browser UI hidden when disabled** — The "Built-in browser" settings toggle and "New Browser Window" menu item are now hidden when `DATAPILOT_DISABLE_BROWSER` is active.
- **Raw body support for API sources** — API source proxy now correctly handles `_rawBody` and `_contentType` parameters for non-JSON payloads (e.g., form-encoded, XML).

## Bug Fixes
- **Bun runtime bundled in packaged app** — The Pi agent subprocess now works correctly in packaged builds. Previously, the missing Bun binary caused a dock icon flash and immediate crash with no session response.
- **Bedrock model defaults and region awareness** — Corrected the default model list for Bedrock connections and added region-aware inference profile resolution (synced from upstream).
- **Model dropdown scrolling** — Dropdown sub-menus are now scrollable when the list exceeds the viewport height.
- **Workspace file access** — Files under the workspace working directory can now be opened correctly.
- **Server lock released on quit** — The server lock file is now properly released when the app quits, with hardened stale lock detection.

## Breaking Changes
- None.
